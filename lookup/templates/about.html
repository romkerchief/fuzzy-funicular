About Weather Lookup

Weather Lookup is a simple web app that lets you check the current weather for any city or location in the world.
Just enter a location on the home page and get instant weather info, powered by OpenWeatherMap.

Features:

  • Search for weather by city or location
  • See temperature, weather conditions, humidity, wind, and the forecast for the locations you chose
  • Easy to use and mobile-friendly

This project was built as a learning exercise using Django and Python.
